# robot
功能：1.进行简单对话 2.计时器功能 3.倒计时提醒（可以实现类似手机代办
事项提醒）4.记事本功能 5.背诵唐诗啥的 6.一些常用操作如程序启动和退出以
及某些对话会有简单语音（所有的计时任务、提醒任务都会汇总到主页信息区显示）
可以进行简单的对话，并记录所有聊天信息，可以选择输出为 sqlite 数据库文件或者 json 文件（我一般当作吐槽和记事本用）
